FAQ

よくあるご質問

お客様にPCやアプリのダウンロードは必要？+

いいえ。スマホのブラウザだけでOK。お渡しするQRをカメラで撮るだけで、マイページが開きます。

既に CarRide や他社サービスを使っているのですが…+

他社の多くは業務効率化ツールなので、くるまカルテ（お客様向け体験）と併用が可能です。

お客様データは安全ですか？+

国内サーバー（Supabase / Vercel）で暗号化して保管。各店舗の情報は完全に分離されます。

既存の Liny / Lステップは活かせますか？+

はい。プロプランで API 連携が可能です。くるまカルテは「タグを付ける」役割を担い、配信文面は今までの Liny シナリオをそのまま使えます。

お客様への押し付け感が心配です+

マイページは「お客様の愛車のページ」として設計しています。ALWAYS WITH YOU カウンターや思い出ギャラリーなど、お客様が見たくなる仕掛けで、自然に開いてくれます。
